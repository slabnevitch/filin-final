Документация для проекта - завода по сжижению природного газа на основаниях гравитационного типа в Арктике

Перевод документации на русский язык для проекта - завода по сжижению природного газа на основаниях гравитационного типа в Арктике.

За основу принят завод СПГ на п-ве Ямал, сам завод модульного исполнения и будет базироваться на платформе морского типа (всего 3 платформы), которые будут транспортироваться от места строительства (верфи). Перевод касался самой технологии СПГ, поэтому специалисты должны были иметь опыт работы по этой тематике.
